Requirements

Windows PC

Access to your Nightreign save folder

Installation

Open File Explorer and navigate to your save directory:

C:\Users\<Your profile name>\AppData\Roaming\Nightreign\<series of numbers>


Open the folder with the series of numbers (this is your personal save folder).

Copy the provided .exe file into this folder.

(Optional) Create a desktop shortcut to the .exe for quick and easy access.

Running the Mod

Double-click the .exe to launch the mod.

Any additional steps or options will be explained within the application.

Notes

Make sure the .exe stays in the save folder for the mod to function correctly.

If you move your saves or reinstall the game, repeat the steps above.
